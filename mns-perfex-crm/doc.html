Create your desktop application by MNS

This tool allows you to easily create a desktop application for Perfex CRM.

Prerequisites

Before you begin, you need to have Node.js and npm (which comes with Node.js) installed on your machine.



How to Install and Run

1. Download the Application:

After purchasing the script, download the `mns-appdriver` file from CodeCanyon. 

Unzip the `mns-appdriver` file to your preferred location.

2. Install Dependencies:

Navigate to the project directory and install the necessary dependencies using npm. 

cd <project directory>
npm install

3. Run the Application:

Now, you can start the Electron application.

npm start

How to Use

  1. Fill in the Perfex CRM URL field with the full URL of your Perfex CRM.
  2. Give your application a name in the Application Name field.
  3. Adjust the Width and Height as desired.
  4. Upload an icon for your application in App Icon.
  5. Choose the Title Bar Style and the Platform you want to generate the application for.
  6. Click Create Application and wait as your application is generated.
